# Print Stickers Workflow

## Steps

1. Login to the system at `/login`

2. Navigate to `/admin/print-batch`

3. Click **"Jana 6 Sticker"** button

4. Verify:
   - 6 QR codes displayed
   - PERKESO logo visible in center of each QR
   - Short ID (8 characters) below each QR

5. Click **"üñ®Ô∏è Cetak A4"** button

6. In print dialog, set:
   - Paper Size: A4
   - Orientation: Portrait
   - Margins: Minimum / None
   - Scale: 100%
   - Background Graphics: ON

7. Print and cut along dashed lines

## Output
- 6 stickers per A4 page
- Each sticker has: Header, QR+Logo, Short ID, Footer
- Dashed cutting guides between stickers

## Notes
- Each "Jana 6 Sticker" generates new UUIDs
- Test scan before mass printing
- Use glossy sticker paper for best results
